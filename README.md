# 哈士奇项目文书中文版

Husky工程是关于新一代的AI和新一代的编程。
具体地说，我们正在实现在计算机视觉领域用更强大高效的模型取代深度学习，辅助把事情拓展到其它领域。

## 不同AI方法之间的比较

Husky与以上诸多方法主要不同在于

- 不预先限制推理时计算的形式

- 从最开始就考虑推理时的计算效率

- 尽可能用完全可解释而且高效的程序解决问题

- 利用新一代编程工具实现"Human in the loop"

- 数学上更正确

详细见[不同AI方法之间的比较](raw/comparison.md)
